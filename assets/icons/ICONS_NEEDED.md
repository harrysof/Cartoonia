# Icons Needed for Cartoonia App

Please add the following icon files to this `assets/icons/` folder:

## Navigation Icons
- `back_arrow.png` - Back navigation arrow (white, 24x24px)
- `settings.png` - Settings gear icon (white, 24x24px)

## Action Icons
- `filter.png` - Filter/funnel icon for content filtering (white, 32x32px)
- `create.png` - Sparkle/magic wand icon for story creation (white, 32x32px)
- `play.png` - Play button for video content (white, 32x32px)
- `send.png` - Send/paper plane icon for chat (white, 24x24px)

## Profile Icons
- `add_profile.png` - Plus icon for adding new profiles (white, 32x32px)
- `avatar_placeholder.png` - Default avatar icon (colorful, 80x80px)

## Video/Content Icons
- `play_overlay.png` - Large play button for video thumbnails (white with transparency, 48x48px)

## Optional Enhancement Icons
- `search.png` - Search/magnifying glass (white, 24x24px)
- `checkmark.png` - Checkmark for selections (orange/teal, 20x20px)
- `close.png` - X icon for closing modals (white, 24x24px)

---

**Style Guide:**
- All icons should be **PNG format with transparency**
- Use **white color** for most icons (they'll show on dark backgrounds)
- Keep icons **simple and minimal** (iOS style)
- Recommended sizes are listed above

**Note:** Currently the app uses inline SVG icons in the HTML. Once you add these PNG files, I'll update the code to use them instead for a more polished look.
